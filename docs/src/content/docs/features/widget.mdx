---
title: 토탐정 위젯
---

![](../../../assets/widget-summary.png)

토탐정 위젯은 설정 페이지를 방문하지 않고도 토탐정의 주요 기능들을 간편하게 사용할 수 있도록 해 주는 위젯입니다. 백준 사이트 내에서는 항상 우측 하단에 보여질 것입니다.

위젯을 우클릭하면 메뉴를 열고 닫을 수 있습니다. 메뉴의 각 아이콘을 클릭하면 각각 설정, 테마 변경, 랜덤 디펜스, 문제 조사, 문제 잠금 기능을 이용할 수 있습니다.

위젯을 좌클릭하여 웹사이트의 상단으로 이동할 수 있습니다. 즉, TOP 버튼과 동일한 기능을 합니다.
